Setup Google OAuth
Untuk sync ke Google Calendar, kamu perlu Google OAuth Client ID. Gratis, sekali setup, sekitar 3 menit.
Cara mendapatkan Client ID:
1. Buka console.cloud.google.com
2. Buat project baru, lalu ke APIs & Services > Credentials
3. Create Credentials > OAuth Client ID > Web application
4. Di Authorized JavaScript origins, tambahkan URL ini
5. Enable Google Calendar API di Library
6. Copy Client ID dan paste di bawah
